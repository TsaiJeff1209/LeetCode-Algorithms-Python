# [28. Implement strStr()](https://leetcode.com/problems/implement-strstr/description/)

## 2018/4/12 beats 99.32 % of python3
### Spend 36 ms
```python
class Solution:
    def strStr(self, haystack, needle):
        """
        :type haystack: str
        :type needle: str
        :rtype: int
        """
        return haystack.find(needle)


```
